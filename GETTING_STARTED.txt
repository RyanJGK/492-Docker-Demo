  SCENARIO 1: Compromised Account
    Look for: "janitor" user impossible travel
    Severity: HIGH
    Action:   Review AI analysis and approve for investigation

  SCENARIO 2: Legacy System Risk
    Look for: "legacy-hmi-1" patch drift
    Severity: CRITICAL (Windows XP, 500+ days)
    Action:   Review upgrade recommendations

    Look for: Telnet (port 23) or FTP (port 21)
    Severity: HIGH

❓ TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Problem: Port 8080 already in use
  Solution: Edit docker-compose.yml, change "8080:8080" to "8081:8080"

  Problem: Services won't start
  Solution: Run ./validate-setup.sh to diagnose issues

  Problem: No alerts shown in dashboard
  Solution: Check if shared/triage.json exists
            View logs: docker-compose logs web

  Problem: Permission errors
  Solution: chmod -R 777 shared/
